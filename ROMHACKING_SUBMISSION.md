Tricolore Crise English Patch

## Version

## Short description

Functionally complete English translation patch for *Tricolore Crise*,
including the full event script, system and battle UI, and required
image-based menus.

## Long description

This unofficial patch translates the complete *Tricolore Crise* event script:
11,224 unique strings covering 14,995 locations on disc. It also covers 1,177
system-text entries and all 43 identified battle-UI entries. The patch includes
in-disc English artwork for the title menu, all 32 status and magic screens,
Options, and system-message plates, so the translated interface works on
compatible Dreamcast hardware as well as emulators.

Version 1.0.2 adds 269 previously absent disc replacements, revises 1,197
existing replacements, and standardizes official/reference spellings for
characters and places throughout the game.

The build validates fixed-width text limits, format tokens, line breaks,
source hashes, image allocations, and patch overlap before producing the BPS
and JSON files. It contains no game image and requires a clean `track03.bin`
with SHA-1 `5608009f040cbed33e4b29da33fc36e29bc24b62`.

This is a functional completion release. Some replacement artwork prioritizes
readability over a perfect visual recreation, and a few non-blocking Japanese
branding, splash, accessory-notice, or portrait-name graphics remain. Japanese
name-entry character grids are intentionally retained because they are input
tables.

## Features

- Complete English event/dialogue translation.
- English system and battle UI.
- English Options, status, magic/skill, title-menu, and system-message art.
- Source-verified BPS and JSON patches with 17,151 validated replacements.
