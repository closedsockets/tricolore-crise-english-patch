# Changelog

## v0.9.0-alpha — 2026-07-09

- Complete English translation for all 11,224 unique event-script strings,
  covering 14,995 locations on disc.
- English coverage for 1,175 system-text entries and all 43 identified battle
  UI strings.
- Fixed-width validation for ASCII text, byte limits, format tokens, line
  breaks, and overlapping replacements.
- Built with [Dreamcast Translation Kit v0.1.0](https://github.com/closedsockets/dreamcast-translation-kit/releases/tag/v0.1.0).
- Title/start-screen and other image-based Japanese UI art remain for a later
  release.
