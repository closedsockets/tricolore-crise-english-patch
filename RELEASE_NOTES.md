# Tricolore Crise English Patch v0.9.0-alpha

First public playtest release of the *Tricolore Crise* English fan translation.

- Complete event script: 11,224 / 11,224 unique strings translated.
- 14,995 event-script locations covered on disc.
- Major system text translated, plus all 43 identified battle UI strings.
- Choices, recurring terminology, color-coded keywords, variables, and line
  breaks are preserved through build-time validation.

## Built with Dreamcast Translation Kit

This patch was created with the open-source
[Dreamcast Translation Kit v0.1.0](https://github.com/closedsockets/dreamcast-translation-kit/releases/tag/v0.1.0).
The toolkit scans a clean GDI dump into a CSV worklist, supports glossary-led
AI-assisted translation batches, validates fixed-width replacements, and
builds BPS patches without distributing game data.

## Remaining work

- Title/start-screen and some other image-based Japanese UI art remain.
- More emulator and hardware playtesting is needed before a 1.0 release.
